### 🐛 알려진 이슈

1. **CORS 제한**
   - 문제: 클라이언트에서 Naver News API, DART API 직접 호출 불가
   - 현재 해결: 샘플 데이터 생성 함수 사용
   - 장기 해결: 백엔드 프록시 서버 필요 (Node.js/Express)

2. **실시간 주가 데이터 부재**
   - 문제: 투자 지표 계산 시 주가 데이터 수동 입력
   - 해결 방안: 한국거래소 API 또는 네이버 증권 크롤링

3. **대용량 데이터 처리**
   - 문제: 여러 연도 데이터 조회 시 성능 저하 가능
   - 해결 방안: Virtual scrolling, lazy loading 도입
